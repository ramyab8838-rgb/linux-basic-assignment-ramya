Section 1

### 1. pwd
Purpose: Shows the current working directory.
Example:
pwd

### 2. ls
Purpose: Lists files and directories.
Example:
ls

### 3. cd
Purpose: Changes the directory.
Example:
cd Documents

### 4. mkdir
Purpose: Creates a new directory.
Example:
mkdir project

### 5. rm -rf
Purpose: Removes files/directories forcefully.
Example:
rm -rf foldername

### 6. ps -ef
Purpose: Shows running processes.
Example:
ps -ef

### 7. top
Purpose: Displays system processes and CPU usage.
Example:
top

### 8. df -h
Purpose: Shows disk usage in human readable format.
Example:
df -h

### 9. history
Purpose: Shows previously executed commands.
Example:
history

### 10. uptime
Purpose: Shows system running time.

Section 3

1. > - overwrites a file, >> - appends content to a file.
2. kill -9 forcefully terminates a process.
3. rm - removes files and directories, rmdir - removes only empty directories.
4. netstat -tulpn - shows active network connections, listening ports, and the process using them.
5. ping - checks connectivity between your system and another server.
